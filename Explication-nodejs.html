Explication de comment installer node.js Installer nodejs Aller sur son editeur de texte et créer un dossier "node_modules" Dans l'éditeur de texte ouvrir un terminal et tapé "npm init" Un fichier package.json/ Tapé npm install express fs installe les modules Node.js express et fs Tapé npm nodemon --save-dev installe le paquet nodemon dans package.json Tapé npm start pour initialiser le serveur Pour finir tapé node app.js pour lancer le script contenu.
